# **App Name**: Habilidade Ativa

## Core Features:

- Hero Section Content Display: Displays the impactful headline, subheadline, main call-to-action button, and key benefits in the hero section.
- Lead Capture Form: Captures user contact information for course enrollment, providing options for WhatsApp redirection or a direct form submission.
- Thematic Content Sections: Presents structured content (Dor, Como Funciona, Sobre a Teacher, Prova Social, FAQ, CTA Final) using elegant cards, image frames, and an accordion for FAQs.
- Subtle UI Animations: Implements soft scroll-reveal effects (fade-in, translateY 8px), subtle background particles in the hero section, and hover effects on buttons for enhanced user engagement.
- Responsive Design: Ensures the landing page layout and visual elements are fully optimized and performant across all devices, including mobile, with reduced visual effects for performance.
- SEO Optimization: Configures the page with appropriate meta titles and descriptions for search engine visibility, specifically for 'Minicurso Gratuito para Destravar o Inglês | Habilidade Ativa'.
- Personalized Follow-up Message Tool: A generative AI tool that crafts initial personalized follow-up messages for newly captured leads, improving conversion to course participation.

## Style Guidelines:

- A light color scheme to evoke a clear, professional, yet magical feel, aligning with the concept of a parchment backdrop. The overall vibe is inspired by an academic, modern wizarding world.
- Background color: Beige parchment (#F5F1E8). This provides a subtle, elegant base reminiscent of old magical scrolls, enhancing readability and warmth.
- Primary color: Soft Gold (#C8A24D). Used for key headings, decorative lines, and subtle highlights. Its moderate brightness and saturation convey elegance and a touch of understated magic, providing a balanced contrast against the light background.
- Accent color: Soft Coral (#E97451). Reserved specifically for Call-to-Action buttons to ensure high visibility and conversion focus. This color provides a vibrant, yet warm contrast to draw immediate attention, while remaining analogous to the primary golden hue for visual harmony.
- Dominant Text color: Deep Forest Green (#1E3D2F). This dark, sophisticated green offers excellent readability on the light parchment background and complements the gold, enhancing the elegant and magical aesthetic.
- Headline font: 'Alegreya', a humanist serif, for its elegant, intellectual, and contemporary feel, suitable for sophisticated titles that hint at the magic of learning.
- Body text font: 'PT Sans', a humanist sans-serif, offering a modern look with a touch of warmth, ensuring readability and balance for all explanatory content.
- Discreet golden particles and subtle glowing effects on titles. Use thin, decorative lines for frames, suggesting ancient magical artifacts. FAQ section icons should be small golden stars, reinforcing the subtle magical theme without infringement.
- A conversion-focused, single-page layout without navigation. Utilizes clearly defined sections for impactful storytelling. Content within sections is structured with elegant cards for features and testimonials, and a collapsible accordion for FAQs.
- Teacher's photo should be presented within a subtle, elegant frame. All content should flow intuitively, minimizing distractions to guide the user towards conversion.
- Ultra-smooth scroll-reveal animations (fade-in + translateY 8px) applied once per element to maintain an elegant, unhurried reveal. Light, discreet particles animate subtly in the hero section background. Buttons feature a gentle golden glow on hover to draw attention and invite interaction. All transitions are set to 300ms ease-out for a refined user experience.
